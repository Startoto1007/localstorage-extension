Local Storage Extension Documentation

A simple extension for Turbowarp that allows access to and modification of localStorage when the project is compiled to HTML.

Available Blocks

set [KEY] to [VALUE]

Stores a value in localStorage associated with the specified key.

  • KEY: The name of the storage location (text)
  • VALUE: The value to store (text)

get [KEY]

Retrieves the value associated with the specified key from localStorage.

  • KEY: The name of the storage location to read from (text)
  • Returns: The stored value, or empty string if the key doesn't exist

delete [KEY]

Removes the specified key and its value from localStorage.

  • KEY: The name of the storage location to delete (text)

key [KEY] exists?

Checks if a key exists in localStorage.

  • KEY: The name of the storage location to check (text)
  • Returns: true if the key exists, false otherwise

storage is empty?

Checks if localStorage is empty (contains no items).

  • Returns: true if localStorage is empty, false otherwise

clear storage

Removes all keys and values from localStorage.

Documentation de l'extension Local Storage

Une extension simple pour Turbowarp permettant d'accéder et de modifier le localStorage lorsque le projet est compilé en HTML.

Blocs Disponibles

définir [CLÉ] à [VALEUR]

Stocke une valeur dans le localStorage associée à la clé spécifiée.

  • CLÉ : Le nom de l'emplacement de stockage (texte)
  • VALEUR : La valeur à stocker (texte)

obtenir [CLÉ]

Récupère la valeur associée à la clé spécifiée dans le localStorage.

  • CLÉ : Le nom de l'emplacement de stockage à lire (texte)
  • Retourne : La valeur stockée, ou une chaîne vide si la clé n'existe pas

supprimer [CLÉ]

Supprime la clé spécifiée et sa valeur du localStorage.

  • CLÉ : Le nom de l'emplacement de stockage à supprimer (texte)

la clé [CLÉ] existe ?

Vérifie si une clé existe dans le localStorage.

  • CLÉ : Le nom de l'emplacement de stockage à vérifier (texte)
  • Retourne : vrai si la clé existe, faux sinon

stockage est vide ?

Vérifie si le localStorage est vide (ne contient aucun élément).

  • Retourne : vrai si le localStorage est vide, faux sinon

vider le stockage

Supprime toutes les clés et valeurs du localStorage.
