Local Storage Extension

Overview

This extension allows you to store and retrieve data directly in the browser's local storage for TurboWarp projects. You can set, get, delete keys, and check if the storage is empty.

Blocks

  • set [KEY] to [VALUE] – Stores a value under a key.
  • get [KEY] – Retrieves the value of a key.
  • delete [KEY] – Deletes a key from storage.
  • key [KEY] exists? – Returns true if a key exists.
  • storage is empty? – Returns true if the storage is empty.
  • clear storage – Removes all keys from storage.
